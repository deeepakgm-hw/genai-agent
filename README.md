# 🚀 AI Career & Interview Feedback Agent

## 📌 Overview

An AI-powered system that analyzes user responses and provides structured feedback for interview preparation.

## ✨ Features

* Confidence analysis
* Strengths & weaknesses detection
* Improved answer suggestions
* Scoring system (out of 10)
* REST API using FastAPI

## 🛠 Tech Stack

* FastAPI
* Python
* AI (Gemini-ready / mock engine)
* Render (Cloud Deployment)

## 🌐 Live API

https://genai-agent-8svq.onrender.com

## 📡 API Endpoints

### GET /

Health check

### GET /ask?q=your_input

Returns AI feedback on user response

## 💡 Example

Input:
"I am good at coding but nervous in interviews"

Output:

* Confidence Feedback
* Strengths
* Weaknesses
* Improved Answer
* Score
